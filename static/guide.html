CCTV AUTOMATION GUIDE

คู่มือการใช้งาน

Web base สำหรับแคปภาพ ดูภาพสด และดึงภาพย้อนหลังจาก NVR

กลับหน้าใช้งาน

ความสามารถ

  • • ดูวิดีโอสดผ่านแท็บ LIVE CCTV
  • • เลื่อน Timeline เพื่อเลือกจุดเวลาใน NVR แล้วย้อนดู 1 นาที
  • • แคปภาพรวมจากกล้อง 18 ช่อง
  • • บันทึกข้ามคืนเวลา 23:00 และ 04:00
  • • บันทึกย้อนหลังแบบข้ามวัน
  • • รวมภาพส่งงานเป็น ZIP ในโฟลเดอร์ exports/

เปิดโปรแกรม

  1. 1. ติดตั้งด้วย ติดตั้งระบบ.command หรือ ติดตั้งระบบ_windows.bat
  2. 2. วางไฟล์ .env ไว้ข้าง app.py
  3. 3. macOS ดับเบิลคลิก เปิดโปรแกรม.command
  4. 4. Windows ดับเบิลคลิก เปิดโปรแกรม_windows.bat

ไฟล์ .env

ขอไฟล์ `.env` จากพี่หมี ห้ามนำไปใส่ GitHub หรือส่งในแชตสาธารณะ

CCTV Autometion/
├── app.py
├── .env
└── static/

ดูวิดีโอสด

  1. 1. เปิดแท็บ LIVE CCTV
  2. 2. เลือกกล้อง
  3. 3. กด เปิดภาพสด
  4. 4. ใช้วันที่และ Timeline ด้านล่างเพื่อเลือกเวลาย้อนหลัง แล้วกด เปิดย้อนหลังจุดนี้
  5. 5. กด หยุดภาพสด เมื่อเลิกดู

ระบบเปิดทีละกล้องผ่าน FFmpeg เพื่อควบคุมการใช้ทรัพยากร Timeline เป็นการเลือกจุดเวลาใน NVR ไม่ใช่การเลื่อนย้อนในสตรีมสดที่กำลังเล่นอยู่

กรณีไม่มีข้อมูล

ถ้า NVR ไม่พบไฟล์ย้อนหลังทุกกล้อง ระบบจะแสดงข้อความ ไม่มีข้อมูลกล้องวงจรปิดในวันที่เลือก แทนคำเตือนรายกล้องจำนวนมาก โดยมักเกิดจากไฟล์ถูก NVR เขียนทับตามระยะเวลาจัดเก็บ
