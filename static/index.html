LOCAL PROJECT

โปรเจกต์นี้ทำอะไรได้บ้าง

  • • แคปภาพรวมจากกล้องที่ตั้งไว้ทั้งหมด 18 ช่อง
  • • เก็บภาพทุกช่องและภาพรวมไว้ในโฟลเดอร์รวมกล้องเดียวกัน
  • • ดูวิดีโอสดจากกล้องที่เลือกผ่านโหมด LIVE CCTV
  • • บันทึกข้ามคืนตามเวลา เช่น 23:00 และ 04:00
  • • ดึงภาพย้อนหลังแบบข้ามวัน และแสดงผลในเว็บ
  • • รวมภาพ ดาวน์โหลด และเปิดดูจากหน้าเว็บได้

เปิดโปรแกรมแบบง่าย

  1. 1. ดับเบิลคลิก เปิดโปรแกรม.command บน macOS
  2. 2. ดับเบิลคลิก เปิดโปรแกรม_windows.bat บน Windows
  3. 3. โปรแกรมจะเปิด Web base ผ่านเบราว์เซอร์
  4. 4. ปิดหน้าต่าง Terminal เมื่อต้องการหยุด

วิธีใช้งานในเว็บ

  1. 1. เลือกแท็บ LIVE CCTV, บันทึกข้ามคืน หรือ บันทึกย้อนหลัง
  2. 2. ตั้งเวลาตามต้องการ ภาพทุกช่องจะถูกเก็บไว้ในโฟลเดอร์รวมกล้อง
  3. 3. กดปุ่มเริ่มงาน แล้วดูความคืบหน้าที่สถานะล่าสุด
  4. 4. หากต้องหยุดระหว่างทาง กด ยกเลิกงาน
  5. 5. ผลลัพธ์จะปรากฏในหัวข้อ ผลลัพธ์ภาพ

ตั้งค่าเครื่องใหม่

  1. 1. คัดลอก .env.example เป็น .env
  2. 2. กรอก Host, Port, Username และ Password ของ NVR
  3. 3. ตรวจสอบว่าติดตั้ง ffmpeg แล้ว
  4. 4. เลือกเปิดด้วยไฟล์ App หรือ Web ตามต้องการ

ไฟล์ .env ใช้เก็บข้อมูลลับและไม่ควรส่งต่อหรืออัปโหลดเข้า Git

โปรแกรมที่ต้องติดตั้ง

  • • Python 3 จำเป็นสำหรับรัน backend
  • • ffmpeg จำเป็นสำหรับดึงภาพจากกล้อง/NVR
  • • Node.js ไม่ต้องติดตั้ง สำหรับรุ่น Web base
brew install python ffmpeg

# Windows PowerShell / Command Prompt
winget install --id Python.Python.3.12 --exact
winget install --id Gyan.FFmpeg --exact

รุ่น Web base ไม่ต้องใช้ Node.js หรือ Electron

Windows: ดับเบิลคลิก ติดตั้งระบบ_windows.bat แล้วเปิด เปิดโปรแกรม_windows.bat

ตำแหน่งไฟล์ผลลัพธ์

captures/วันที่/เวลา/ จะมีโฟลเดอร์ รวมกล้อง และไฟล์ manifest.json โดยภาพกล้องแต่ละช่องอยู่ในโฟลเดอร์รวมกล้องเดียวกัน

ถ้าบางกล้องไม่มีภาพ ระบบจะแสดงเป็น “เสร็จบางส่วน” พร้อมระบุหมายเลขกล้องที่มีปัญหา ส่วนภาพของกล้องอื่นยังใช้งานได้ตามปกติ
