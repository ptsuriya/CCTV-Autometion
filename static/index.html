LOCAL PROJECT

โปรเจกต์นี้ทำอะไรได้บ้าง

  • • แคปภาพรวมจากกล้องที่ตั้งไว้ทั้งหมด 18 ช่อง
  • • เลือกกล้องสำหรับจัดเก็บเป็นรูปเดี่ยวได้
  • • บันทึกข้ามคืนตามเวลา เช่น 23:00 และ 04:00
  • • ดึงภาพย้อนหลังแบบข้ามวัน และแสดงผลในเว็บ
  • • รวมภาพ ดาวน์โหลด และเปิดดูจากหน้าเว็บได้

เปิดโปรแกรมแบบง่าย

  1. 1. ดับเบิลคลิกไฟล์ เปิดโปรแกรม.command
  2. 2. โปรแกรมจะเปิดเซิร์ฟเวอร์และเปิดเว็บให้อัตโนมัติ
  3. 3. ถ้าเปิดอยู่แล้ว ไฟล์เดิมจะเปิดหน้าเว็บเดิมให้
  4. 4. ปิดหน้าต่าง Terminal เมื่อต้องการหยุดโปรแกรม

วิธีใช้งานในเว็บ

  1. 1. เลือกแท็บ บันทึกข้ามคืน หรือ บันทึกย้อนหลัง
  2. 2. ตั้งเวลาและเลือกกล้องรูปเดี่ยวตามต้องการ
  3. 3. กดปุ่มเริ่มงาน แล้วดูความคืบหน้าที่สถานะล่าสุด
  4. 4. หากต้องหยุดระหว่างทาง กด ยกเลิกงาน
  5. 5. ผลลัพธ์จะปรากฏในหัวข้อ ผลลัพธ์ภาพ

ตั้งค่าเครื่องใหม่

  1. 1. คัดลอก .env.example เป็น .env
  2. 2. กรอก Host, Port, Username และ Password ของ NVR
  3. 3. ตรวจสอบว่าติดตั้ง ffmpeg แล้ว
  4. 4. เปิดด้วย เปิดโปรแกรม.command

ไฟล์ .env ใช้เก็บข้อมูลลับและไม่ควรส่งต่อหรืออัปโหลดเข้า Git

โปรแกรมที่ต้องติดตั้ง

  • • Python 3 จำเป็นสำหรับรันเซิร์ฟเวอร์
  • • ffmpeg จำเป็นสำหรับดึงภาพจากกล้อง/NVR
  • • Node.js ไม่จำเป็น สำหรับเวอร์ชันปัจจุบัน
brew install python ffmpeg

# Node.js เป็นตัวเลือก เฉพาะกรณีจะพัฒนาหน้าเว็บแบบ React เพิ่มเติม
brew install node

# Windows PowerShell / Command Prompt
winget install --id Python.Python.3.12 --exact
winget install --id Gyan.FFmpeg --exact

หรือดับเบิลคลิกไฟล์ “ติดตั้งระบบ.command” เพื่อให้โปรแกรมตรวจสอบและติดตั้ง Python กับ ffmpeg ผ่าน Homebrew

Windows: ดับเบิลคลิก ติดตั้งระบบ_windows.bat แล้วเปิดด้วย เปิดโปรแกรม_windows.bat

captures/วันที่/เวลา/ จะแยกเป็นโฟลเดอร์ รวมกล้อง, รูปเดี่ยว และไฟล์ manifest.json
